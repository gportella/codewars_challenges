# DESCRIPTION:

Given a string `s`, composed of lowercase letters, you can perform the following operation on the string:

- Choose any two characters in the string that are the same and delete them

**Task**: Find the minimum lexigraphical string that occurs out of all possibilities after performing this operation on s as many times as possible.

 Example

 ```
 solve("czaccb") -> "czab"
 solve("geeegfffgdadd") -> "efgad"
 solve("cdeddaeeb") -> "cdaeb"
```
